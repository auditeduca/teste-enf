## 1. Purpose

Definir goals (metas) como alvos mensuráveis que guiam o planejamento de intervenções.

## 2. Goal Model

```
Goal
├── noc_code (PK, e.g., "1101" — Tissue Integrity)
├── current_value (e.g., 2 — substantially compromised)
├── target_value (e.g., 4 — mildly compromised)
├── delta = target - current (+2)
├── time_horizon (e.g., 72h)
├── priority (1-5)
├── linked_nanda_code (what diagnosis this addresses)
├── linked_nic_codes[] (what interventions achieve this)
└── success_criteria
```

## 3. Goal Setting in Reasoning

```
NANDA 00047 → NIC 3540 → NOC 1101
Current: 2/5 → Target: 4/5 → Delta: +2 → Horizon: 72h
    ↓
Plan: Apply NIC 3540 q2h, reassess NOC at 24h, 48h, 72h
    ↓
Contingency: If NOC not improving at 24h → escalate to NIC 2250
```

## 4. NOC Scale (1-5)

| Score | Label | Meaning |
|-------|-------|---------|
| 1 | Severely compromised | Worst state |
| 2 | Substantially compromised | Significant impairment |
| 3 | Moderately compromised | Partial impairment |
| 4 | Mildly compromised | Near-normal |
| 5 | Not compromised | Optimal |

## 5. NIS Implementation

Goals are stored in `ni_planner.plans` and tracked via `ni_memory.outcomes`:

```
plan.goal_noc_code = "1101"
plan.expected_noc_delta = +2
outcome.measured_at = T+24h
outcome.outcome_value = 3 (improved from 2 to 3)
outcome.outcome_type = "improved"
```
