## 1. Purpose

Definir como o NIS interoperabilidade via HL7 v2 — o padrão legacy mais usado em hospitais brasileiros.

## 2. HL7 v2 Messages

| Message Type | Trigger | NIS Use |
|-------------|---------|---------|
| ADT^A01 | Patient admission | Create patient context |
| ADT^A03 | Patient discharge | Close episode |
| ADT^A08 | Patient update | Update demographics |
| ORU^R01 | Observation result | Ingest labs/vitals → observations |
| RDE^R05 | Pharmacy order | Medication tracking |
| MDM^T02 | Document notification | Clinical document import |

## 3. NIS HL7 Adapter

```
HL7 v2 Message (pipe-delimited)
    ↓ Parse (hl7-parser)
Mapped to NIS entities:
  MSH → message metadata
  PID → ni_user.patients
  OBR/OBX → ni_clinical.observations
  PV1 → encounter context
    ↓ Store in ni_interop.incoming_messages
    ↓ Validate against schema
    ↓ Feed to reasoning engine
```

## 4. NKOS Reference
